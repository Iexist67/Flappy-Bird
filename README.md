# Flappy-Bird
A simple replication of the popular game, flappy-bird. Try to dodge pipes as a bird!

**Here is a "try me"**

  link: https://iexist67.github.io/Flappy-Bird/

**Features:**


  . There are difficulty modes that increase or decrease the gap inbetween the pipes
  
  . A session scoreboard. A scoreboard that only saves in a user session

  . Everything in the background moves and loops around
  
  . The pipes will increase speed every time you score a point
  
  . To make flappy jump, just press the space, w, up arrow, or trackpad to make flappy jump
  
  . Added a custom font to the files for the text in the game.

**How to run:**

  . Use any web browser
  
  . Click the link and start playing!

**Technicals:**

  I chose to use javascript paired with html and css to create my flappy bird game because it was the languages I was familiar with and comfortable to create this game. I decided not to use classes and objects in this game because I hadn't grasped the idea of classes and objects, so the pipes just loop around on the screen. I also didn't use a canvas for this project because it was easier to use a div element to add images to my characters.
